# MS-02 Ultra

The MS-02 Ultra is a MINI PC that represents a comprehensive upgrade from the MS-01.  
This guide provides a brief introduction to the MS-02 Ultra and includes important information about memory compatibility, usage, and other key details.

**When purchasing memory, please be sure to consult the official [Memory Shipping List](/Docs/English/02-Ultra-Memory.md) and the [QVL list](/Docs/English/02-Ultra-Memory.md).**
